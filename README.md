# Ship-prediction

### 项目来源

- 题目来源于`[datafountain](https://www.datafountain.cn/competitions/545)`平台中的竞赛题：船舶航行轨迹预测
- 在船舶航行轨迹预测赛题中，目标是基于历史船舶轨迹数据，预测未来一段时间内船舶的航行轨迹。
  - 比赛提供的数据集包含了大量的船舶轨迹记录，包括位置坐标、时间戳、速度等信息。
  - 参赛者需要利用这些数据，建立有效的预测模型，并预测出未来船舶的轨迹。具体参考网站

### 文件介绍

- `train.csv`：训练集
- `test.csv`：测试集
- `LSTM.ipynb`：模型代码
